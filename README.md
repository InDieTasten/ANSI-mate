# ASCII-mate
A CLI tool with mouse support for drawing ascii art and animations

## Prerequisites
- /bin/stty
- /bin/tput
- /bin/sleep
- luajit or lua

## Usage
```bash
git clone git@github.com:InDieTasten/ASCII-mate.git
cd ASCII-mate
luajit main.lua
```
